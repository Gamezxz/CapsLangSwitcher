How it works

1

Tap the Caps Lock key — no need to hold it

2

A CGEventTap grabs it at the HID level and swallows the real caps-lock toggle entirely

3

The app calls macOS's Text Input Source API directly, in-process — your input source switches on the spot

Get started in 2 minutes

  1. Download CapsLangSwitcher.app.zip from Releases and move it to Applications
  2. Double-click to open — it's signed and notarized, so no Gatekeeper warnings
  3. Allow Accessibility access when prompted (System Settings → Privacy & Security → Accessibility)
  4. Turn off any existing Caps Lock shortcut in System Settings → Keyboard → Keyboard Shortcuts → Input Sources
  5. Tap Caps Lock and enjoy instant switching
